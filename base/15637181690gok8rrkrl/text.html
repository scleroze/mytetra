FFTW3

(default: gfortran and gcc)

1) tar zxvf fftw-3.3.4.tar.gz

2) cd fftw-3.3.4

3) ./configure --enable-threads --enable-openmp --enable-avx

./configure （For gromacs ./configure --enable-threads --enable-float ）

4) make

5) sudo make install


*** fftw library adress ***

/usr/local/lib/libfftw3.a

/usr/local/lib/libfftw3_threads.a

/usr/local/lib/libfftw3_omp.a (openmp)

***********


Recompile: make clean, ./configure ..., make and sudo make install.
